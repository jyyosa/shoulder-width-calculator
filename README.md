# 肩宽推算器

> 基于 **GB/T 10000-2023** 中国成年人人体尺寸最新国家标准，同时推算水平肩宽与过肩线肩宽。

**在线使用**：https://jyyosa.github.io/shoulder-width-calculator/

Made by [@jyyosa](https://github.com/jyyosa)

---

## 功能简介

这是一个纯前端 HTML 工具，无需安装，打开网页即可使用。它同时推算两种肩宽：

| 类型 | 定义 | 用途 |
|------|------|------|
| **水平肩宽** | 两肩端点(SP)之间的水平直线距离 | 体型分析、号型对照 |
| **过肩线肩宽** | 皮尺贴着肩膀表面，从一侧肩点经后颈点(BNP)到另一侧肩点的曲线长度 | 裁缝制版、定制服装 |

两者的差值主要来自皮尺绕行颈后隆起的**额外弧长**（通常约 3.5~5.0cm）。

## 算法原理

### 核心思路：身高主导 + 差异感知微调

人体各部位比例存在个体差异，简单的加权平均会导致"偏矮的数据拉低整体结果"。本算法采用**身高法为主导**，其他参数只在**与身高法差异较小**时才纳入微调。

### 1. 身高法（权重 80%~95%）

```
水平肩宽 = 身高 × 比例系数
```

| 性别 | 系数 | 来源 |
|------|------|------|
| 男 | 0.2235 | GB/T 10000-2023 P50 反推 |
| 女 | 0.2233 | GB/T 10000-2023 P50 反推 |

**为什么身高法最可靠？**
- 身高与肩宽的相关系数在人体各维度中最高（r > 0.85）
- 身高测量误差最小，不受呼吸、姿态影响

### 2. 胸围法（辅助，自动降权）

```
胸围估计 = 胸围 × 系数
chestDiff = |胸围估计 - 身高法估计|

if chestDiff < 2cm:   权重 20%
if chestDiff < 4cm:   权重 10%
else:                 权重 5%
```

胸围法对健美/瘦弱体型敏感。当用户胸围异常（如健身达人胸围大、瘦弱者胸围小）时，自动降低权重避免拉偏。

### 3. BMI 体型修正（整体缩放 ±3%）

| BMI 范围 | 修正系数 | 说明 |
|----------|----------|------|
| < 18.5 | 0.97 | 偏瘦，骨架相对窄 |
| 18.5 ~ 24 | 1.00 | 正常，无需修正 |
| > 24 | 1.03 | 超重，骨架相对宽 |

### 4. 臂长 / 颈围验证（差异大时自动弃用）

```
臂长估计 = 外臂长 × 0.680
if |臂长估计 - 当前值| < 5cm:
    纳入计算，权重 5%
else:
    仅作参考显示，不纳入计算

颈围估计 = 颈围 × 1.018
if |颈围估计 - 当前值| < 5cm:
    纳入计算，权重 3%
else:
    仅作参考显示，不纳入计算
```

**为什么需要差异检测？**
- 臂长受测量方式影响大（内臂/外臂混淆、肩点定位偏差）
- 颈围与肩宽的相关性较弱，只应在合理范围内做微调

### 5. 过肩线肩宽

```
过肩线肩宽 = 水平肩宽 + 颈后弧长增量 + 肩斜修正

颈后弧长增量 = clamp(3, 6, 颈围 × 0.115)  // 约 3.5~5.0cm
肩斜修正 = 基础值 × 0.12  // 约 0.3~0.4cm
```

### 6. 身高肩宽指数分型（GB/T 10000-2023）

```
身高肩宽指数 = 肩宽 / 身高 × 100
```

| 分型 | 男性 | 女性 |
|------|------|------|
| 窄肩型 | < 21.5 | < 20.0 |
| 标准型 | 21.5 ~ 23.5 | 20.0 ~ 22.0 |
| 宽肩型 | > 23.5 | > 22.0 |

## 使用方法

### 必填项

- **身高**：站立赤脚，头顶至脚跟（cm）
- **胸围**：腋下水平绕胸部最丰满处（cm）

### 推荐项

- **体重**：用于 BMI 体型修正，提升精度
- **颈围**：喉结下方水平绕颈一周，用于过肩线推算

### 选填项

- **外臂长**：肩点(SP)沿手臂外侧至腕横纹，手臂自然下垂
- **腰围**：躯干最细处水平围量，用于判断 H/V 型躯干

### Kigurumi 头壳参数（可选）

输入头壳围、头壳脸宽、真人头长后，可分析：
- 戴头壳后的头身比
- 与二次元角色的匹配度
- 真人身体 vs 动漫角色身体比例对比
