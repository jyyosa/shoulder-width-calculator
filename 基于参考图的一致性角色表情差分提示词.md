# 基于参考图的一致性角色表情差分生成指南 + 提示词库

> **核心原则**：画风零偏移、五官零变形、服饰零改动、构图零变化，**仅修改面部表情**
> 
> **适用工具**：Stable Diffusion (img2img/ADetailer/ControlNet) / FLUX / Midjourney Image Prompt / NovelAI / 可图等
> 
> **执行前提**：你已有角色设定图（图一），需要以此为基准生成6表情差分

---

## 一、执行方法总览（推荐按此流程操作）

### 方法 A：img2img 局部重绘（最推荐，精度最高）

1. **将图一上传到 img2img**
2. **用画笔涂抹面部区域**（眉毛+眼睛+嘴巴+脸颊，必须覆盖眉毛！）
3. **保持原提示词不变**，只替换表情关键词
4. **固定以下参数**：
   - Seed：与原图一致（点击绿色箭头从原图复制）
   - Sampler：DPM++ 2M Karras
   - Steps：25
   - CFG：7
   - **Denoising Strength：0.35 - 0.45**（关键！越低越像原图，0.4是最佳平衡点）
   - Mask Blur：4-8
   - Mask Mode：Inpaint masked（仅重绘涂抹区域）
   - Mask Content：Original
5. **开启 ControlNet**（可选但强烈建议，用于锁住姿势和轮廓）：
   - Unit 0：ControlNet → Canny / Lineart → 上传图一 → 权重 0.8-1.0
   - Unit 1（可选）：ControlNet → OpenPose → 上传图一 → 权重 0.8-1.0
6. **生成**，只改表情，其余不变

### 方法 B：ADetailer 自动修脸（最快，适合批量）

1. **图一作为基础图**放到文生图或图生图
2. **复制原图完整提示词**（保持角色描述、服装、姿势完全一致）
3. **开启 ADetailer**：
   - 模型选择：`anime_face_yolov8.pt`（动漫）或 `face_yolov8n.pt`（写实）
   - 在 ADetailer Prompt 中**只写表情词**，如 `angry, frowning`
   - Denoising Strength：0.4-0.5
   - Mask Blur：4-8
4. **固定 Seed**，其余参数与原图一致
5. **每次只改 ADetailer 中的表情词**，批量生成6个表情

### 方法 C：Seed 固定 + 提示词替换（最简单，但姿势可能有微变）

1. 记录图一的 **Seed、Sampler、Steps、CFG、Size**
2. 文生图或图生图，输入**原提示词**，仅替换表情词部分
3. 保持 Seed 和所有参数**完全一致**
4. 生成

> **提示**：如果你的 AI 工具支持 **Image Prompt（图生图以图一为参考）**，直接上传图一 + 下方面提示词，并设置**高参考权重**（如 `--iw 2` 或 `strength 0.85`），效果最佳。

## 三、随机生成模板（随缘出不同角色，6表情一次/分批生成）

### 模板 1：Stable Diffusion / Forge 随机表情差分（固定图一 + 随机新角色）

```
[图一作为参考图/ControlNet输入], 
masterpiece, best quality, 1girl, solo, 
[保留原角色描述], 
{expressionless|neutral|calm}, {smile|happy|light smile|grin}, 
{angry|frowning|irritated|pouting}, {crying|tears|sobbing|tearful eyes}, 
{surprised|shocked|wide eyes|mouth agape}, {blushing|embarrassed|shy|nose blush}, 
consistent face, same hairstyle, same outfit, same pose, 
upper body, portrait, from front, looking at viewer, 
white background, simple background, 
ultra detailed, anime style, same art style as reference, 
--seed [原图Seed] --strength 0.4
```

> **使用方法**：不支持花括号 `{}` 随机语法的工具，手动替换为 6 个表情词分别生成 6 张。

### 模板 2：FLUX 高保真随机表情网格（一次性出 6 格）

```
Using the character in the reference image, generate a 2×3 expression grid 
with exactly the same character design, hairstyle, clothing, and pose in all 6 panels. 
Only change the facial expression in each panel:

Panel 1: neutral expressionless face
Panel 2: gentle smile, happy
Panel 3: angry, frowning, pouting
Panel 4: crying, tears streaming down
Panel 5: surprised, wide eyes, mouth open
Panel 6: blushing, embarrassed, shy

Maintain 100% character consistency across all panels. 
White background, anime style, ultra detailed.
```

### 模板 3：Midjourney / Niji 随机表情差分（Image Prompt 模式）

```
[上传图一作为 Image Prompt] 
Expression sheet of the same character, 
6 different expressions in 2×3 grid: 
neutral, smiling, angry, crying, surprised, blushing. 
Identical character design, hairstyle, clothing, and pose. 
Only face changes. White background, anime style, clean linework, 
--ar 16:9 --niji 6 --iw 2 --style raw
```

### 模板 4：可图 / 通义万相 / 文心一格 中文提示词

```
参考图角色，保持完全一致的发型、服装、姿势和画风，
仅修改面部表情为以下6种：
1. 无表情  2. 微笑  3. 生气  4. 哭泣  5. 惊讶  6. 害羞

二次元动漫风格，白色背景，上半身特写，高清细节，
角色一致性100%，仅面部情绪变化。
```

## 六、快速执行清单

| 步骤 | 操作 | 关键设置 |
|------|------|---------|
| 1 | 上传图一到 img2img | 保持原图分辨率 |
| 2 | 涂抹面部区域（眉毛+眼+嘴+脸颊） | 必须覆盖眉毛！ |
| 3 | 输入原角色完整提示词 | 复制原图提示词 |
| 4 | 只替换表情词为上方6个之一 | 其余完全不变 |
| 5 | 固定 Seed | 与原图一致 |
| 6 | Denoising Strength | **0.35-0.45** |
| 7 | 开启 ControlNet Canny | 权重 0.9（锁住轮廓） |
| 8 | 生成6张，每张只改表情词 | 保存到同一文件夹 |
| 9 | 用 Photoshop / Eagle 合成 2×3 网格 | 对齐后导出 |

---

> **总计**：4 个随机模板 + 10 组 × 6 表情 = **64 个提示词**
> 
> 所有提示词共享同一结构：`[原角色描述] + [表情词] + 固定构图/背景/质量词`
> 
> 只需替换 `[原角色描述]` 和 `[原图Seed]` 即可直接生成你的角色表情差分。
